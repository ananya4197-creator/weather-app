# Weather App 🌦️

A simple and user-friendly Weather Application built using Python and Tkinter. This application allows users to check current weather information for any city through a graphical user interface.

## Features
- Search weather by city name
- Displays temperature, weather condition, humidity, and wind speed
- Easy-to-use GUI built with Tkinter
- Real-time weather data using API integration
- Clean and responsive interface

## Technologies Used
- Python
- Tkinter
- Requests Library
- Weather API
